Speech segments

  • Mark only when this speaker is speaking directly to the recording — not background or side conversations.
  • Start/end times should be accurate to ±10ms.
  • Background speakers, TV, and other noises are not speech — no interval there.
  • If audio is completely unclear for a turn, set its transcript to [unintelligible].

Transcript rules

  • Spelled words: caps with spaces — "My name is spelled A L E X".
  • IDs / numbers: numerical — "My ID is 23582", "My zip is 97229".
  • Acronyms: all caps, no periods — "FBI", "NSA".
  • Emails: proper format (john.example@gmail.com); if the speaker spells out letters/numerals, follow the spelling rule instead.
  • Punctuation: proper sentences — periods, commas, !, ?.
  • Proper nouns & brand names: capitalize.
  • Disfluencies: include "um", "uh", etc.
  • Language switching mid-sentence (e.g. English + Spanish): reflect both accurately.
  • Non-speech events / emotions: tag inline — [laughs] [sighs] [groans] [sarcastic] [excited] [yell] [sob] [sing] [whispers] [chuckles].
